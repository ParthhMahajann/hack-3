⚙️ Scoring Methodology

Type: Weighted multi-parameter composite score (0–100+)

Classification:

GREEN 0-29 Routine follow-up YELLOW 30-59 Increased monitoring, inform ANM RED 60-79 Urgent referral, alert ANM within 1 hour PURPLE 80+ EMERGENCY — SMS to Block Officer, immediate referral

Trend Detection: Linear regression slope on BP history (≥3 readings). Rising BP (+4 mmHg/visit) triggers one-tier escalation even before threshold breach.

🤖 ML 30-Day Predictive Model

Model: Logistic Regression (interpretable, no black box)

Formula: P(adverse) = σ(w₁x₁ + w₂x₂ + ... + wₙxₙ + b)

Output: 30-day probability of adverse outcome + top 3 contributing factors

Calibration: NFHS-5 India prevalence rates (~8.5% maternal complications, ~9% child severe malnutrition)

Reference: Rana et al., BMC Pregnancy & Childbirth, 2023 DOI: 10.1186/s12884-023-05387-5

Feature Weights (Maternal):
BP slope: +0.88 · Hemoglobin: -0.82 · Systolic BP: +0.71 · Prev. complications: +0.70 · Diastolic BP: +0.65 · Proteinuria: +0.60 · Missed ANC: +0.55 · Oedema: +0.45 · BMI: -0.31 · Age: +0.28

✅ Validation & Testing

🔄 Offline Sync Architecture

Strategy: CRDT-inspired delta sync with last-write-wins per field

Reference: Shapiro et al., INRIA RR-7506, 2011

📱 ASHA Phone (IndexedDB + Service Worker)
↕️ Background Sync (priority: PURPLE → RED → YELLOW → GREEN)
🌐 FastAPI Server (SQLAlchemy async)
↕️ Conflict detection + ANM review queue
🏥 Block Officer Dashboard (Charts + HMIS Export)

Offline Guarantee: All data saved to IndexedDB first (offline-first). Risk scoring computed client-side using identical thresholds. Syncs when connectivity returns via Background Sync API.
